 # KAPE

Plataforma de comercio electrónico enfocada en conectar directamente a productores de café mexicanos con consumidores y empresas, eliminando intermediarios para promover un comercio más justo, transparente y rentable.

---

## Descripción

**KAPE** es una plataforma digital de comercio electrónico que busca transformar la comercialización del café en México mediante un modelo de venta directa.

El proyecto está dirigido inicialmente a los caficultores de **Hueytamalco, Veracruz**, permitiéndoles ofrecer sus productos a clientes finales, cafeterías, restaurantes y empresas, obteniendo un pago más justo por su trabajo.

Además, la plataforma coordina el proceso de compra y envío, brindando una experiencia segura tanto para productores como para consumidores.

---

## Objetivo

Desarrollar una plataforma de comercio electrónico que elimine el intermediarismo en la venta de café, permitiendo a los productores comercializar directamente sus productos, incrementar sus ingresos y ampliar su alcance hacia mercados nacionales e internacionales, promoviendo al mismo tiempo el consumo de café mexicano de alta calidad.

---

## Propuesta de valor

KAPE ofrece una plataforma segura, intuitiva y accesible que conecta directamente a caficultores con consumidores y empresas, garantizando:

- Precios justos para los productores.
- Café mexicano de alta calidad para los consumidores.
- Coordinación del proceso de compra y envío.
- Comercio transparente y sostenible.
- Impulso al desarrollo económico de las comunidades cafetaleras.

---

# Funcionalidades

La plataforma cuenta con las siguientes secciones:

- Inicio (Home)
- Catálogo de productos
- Detalle del producto
- Carrito de compras
- Checkout
- Inicio de sesión
- Registro de usuario
- Perfil del usuario
- Reseñas

# Trabajo colaborativo

KAPE está siendo desarrollado por un equipo de 9 integrantes** utilizando Git como sistema de control de versiones.
Cada integrante desarrolla una funcionalidad en su propia rama, posteriormente integra sus cambios en la rama develop y, una vez validados, se fusionan a la rama main.

# Instalación

Clona el repositorio:

```bash
git clone URL_DEL_REPOSITORIO
```

Entra a la carpeta del proyecto:

```bash
cd NOMBRE_DEL_REPOSITORIO
```

Si se trata del frontend, abre el archivo:

```text
index.html
```

También puedes utilizar la extensión Live Server de Visual Studio Code.
